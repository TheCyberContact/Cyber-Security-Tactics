Check your kernel for known exploits

cat /proc/version

Gobuster brute forcing URIs

gobuster -v -u https://wesitename -w /path/to/dictionary

Kali has good dictionaries in /usr/share/wordlists/

Dirbuster GUI brute forcing URIs

To get a GUI version of gobuster use the command:

root@kali:~# dirbuster

Sudo to root

Quick web HTTP share of current directory

or

Use wget to download a file

wget https://address/file

Use curl to download a file

curl https://address/file.sh --output file.sh

Get the certificate from a server

openssl s_client -connect 10.0.0.1:443

Use grep to extract emails from files

grep -E -o "\b[A-Za-z0-9._%+-]+@[A-Za-z0-9.-]+\.[A-Za-z]{2,6}\b" file.txt

Use grep to extract IP addresses from files

grep -E -o "(25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)\.(25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)\.(25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)\.(25[0-5]|2[0-4][0-9]|[01]?[0-9][0-9]?)" file.txt

Use grep to extract key words

grep -i "user\|invalid\|authentication\|login|pass\|passw" file.txt

Extract md5 hashes from files

egrep -oE '(^|[^a-fA-F0-9])[a-fA-F0-9]{32}([^a-fA-F0-9]|$)' *.txt | egrep -o '[a-fA-F0-9]{32}' > hashes.txt

Use grep to extract HTTP URL's from files

grep http | grep -shoP 'http.*?[" >]' *.txt > http.txt

Use grep to extract HTTPS URL's from files

grep -E '(((https)[.:][^ >" ]*|www.[-a-z0-9.]+)[^ .,; >">):]' *.txt > https.txt

Disable pings

iptables -A INPUT -p icmp -m icmp --icmp-type any -j DROP

iptables -A OUTPUT -p icmp -j DROP

Flush IPTABLES

iptables --flush

Reverse text content in a file

BASE64 encode a string

echo 'string' | base64

BASE64 decode a string

echo 'base64string==' | base64 --decode

Get a line count
